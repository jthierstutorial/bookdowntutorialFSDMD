Bookdown: Flexible Document Creation in RStudio

Chapter 10 Cross-references

Cross-references make it easier for your readers to find and link to elements in your book. In Bookdown, you can create cross-references for chapters, sub-chapters, figures, and tables.

10.1 Chapters and Sub-Chapters

There are two steps to cross-reference any heading:

  1. Label the heading:
    Add a label by placing {#your-label} at the end of the heading:

    # Hello World {#nice-label}

    • If you donâ€™t add a label, Bookdown will automatically generate one based on the heading title. For example, # Hello World would become # Hello World {#hello-world}.

    • To label an unnumbered heading, use {.unnumbered} at the end:

      # Hello World {-#nice-label}

  2. Reference the labeled heading:
    To reference a labeled heading anywhere in the text, use \@ref(label). For example:

    Please see Chapter \@ref(cross) for more details.

    • If you prefer text as the link instead of a numbered reference, you can use Markdown syntax:

      [Click here to read about cross-references](#cross)

10.2 Captioned Figures and Tables

Figures and tables with captions can also be cross-referenced from elsewhere in your book.

10.2.1 Cross-Referencing Figures

To cross-reference a figure, add a chunk label and set the fig.cap option for the caption. Bookdown will automatically label the figure with fig:chunk-label.

Example:

par(mar = c(4, 4, .1, .1))
plot(pressure, type = 'b', pch = 19)

Figure 10.1: Here is a nice figure!

You can reference this figure using \@ref(fig:nice-fig). For example:

See Figure \@ref(fig:nice-fig) for more details.

10.2.2 Cross-Referencing Tables

To cross-reference a table, add a chunk label to the code chunk where you create the table with knitr::kable() and include a caption. Bookdown will automatically label the table with tab:chunk-label.

Example:

knitr::kable(
  head(pressure, 10), caption = 'Here is a nice table!',
  booktabs = TRUE
)

10.3 Summary

In this chapter, we covered how to create cross-references for headings, figures, and tables. Using cross-references helps readers navigate your document easily and allows you to create a more interactive and organized book.
