Bookdown: Flexible Document Creation in RStudio

Chapter 8 LaTeX in Bookdown

Bookdown offers powerful support for LaTeX, allowing you to seamlessly integrate any LaTeX packages you need into your documents. Whether you’re working with mathematical equations, theorems, lemmas, proofs, or other advanced features, this tutorial will guide you through the essentials of using LaTeX in Bookdown, showing you how to effectively incorporate and reference these elements.

8.1 Including LaTeX Packages in Bookdown

One of the powerful features of Bookdown is its seamless integration with LaTeX, allowing you to include any LaTeX package that suits your needs. This flexibility is especially useful when working with advanced mathematical notations, custom formatting, or specialized content.

8.1.1 Using a preamble.tex File

To include LaTeX packages, you can create a preamble.tex file and link it in your _output.yml file. The preamble.tex file is processed before the document is rendered, making it the ideal place to load additional LaTeX packages.

  1. Create a preamble.tex File
    Add any LaTeX package you want to use in your document. For example here is a list of packages that are typically used by students first introduced to LaTeX:

    % preamble.tex
\usepackage{booktabs}
\usepackage{amssymb}
\usepackage{amsmath}
\usepackage{graphicx}
\usepackage{wasysym}
\usepackage{amsthm}
\usepackage{multirow}
\usepackage{epsf}
\usepackage{tikz}
\usepackage{cancel}
\usepackage{hyperref}
\usepackage{gensymb}
\usepackage{marvosym}

  2. Link preamble.tex in _output.yml
    Update your _output.yml file to include the preamble.tex file for PDF output:

    bookdown::pdf_book:
  latex_engine: xelatex
  includes:
    in_header: preamble.tex

8.1.2 Example: Using AMSMath

Once the amsmath package is included, you can use its features, such as environments for aligned equations:

\begin{align}
  a + b &= c \\
  x - y &= z
\end{align}

8.1.3 Adding Multiple Packages

You can include as many packages as you need in the preamble.tex file. For example:

% preamble.tex
\usepackage{amsmath, amssymb} % Math packages
\usepackage{color} % For colored text
\usepackage{hyperref} % For hyperlinks
\usepackage{booktabs} % For enhanced table formatting

8.1.4 Tips for Using LaTeX Packages

  1. Ensure Compatibility: Some LaTeX packages may conflict with each other. If errors occur, review the documentation for each package to ensure compatibility.

  2. Use the Right LaTeX Engine: Some packages (e.g., those using custom fonts) may require a specific LaTeX engine, such as xelatex or lualatex. Specify the engine in _output.yml:

    bookdown::pdf_book:
  latex_engine: xelatex

  3. Debugging Errors: If your document fails to compile, check the error message in the console or look at the .log file generated by LaTeX for troubleshooting.

By including custom LaTeX packages, you can extend the functionality of your Bookdown project to suit your unique requirements, whether it’s for academic papers, technical documents, or books.

8.2 Mathematical Equations

LaTeX is widely used for formatting mathematical equations. Bookdown makes it easy to include both inline and display-style equations.

8.2.1 Inline Equations

Use $...$ to include inline math equations within your text. The following code:

The formula for the area of a circle is $( A = \pi r^2 )$, where $( r )$ is the radius.

Is then displayed as follows:

The formula for the area of a circle is \(( A = \pi r^2 )\), where \(( r )\) is the radius.

8.2.2 Display Equations

For equations that you may want to have on their own line you have two options. You can surround your LaTeX code $$...$$ or alternatively use the LaTeX equation environment.

Here we see the usage of $$...$$

$$
E = mc^2
$$

Which gives us:

\[ E = mc^2 \]

Here we are using the equation environment with a label:


\begin{equation}
  E = mc^2
  \label{eq:einstein}
\end{equation}

Which outputs as such:

\[\begin{equation} E = mc^2 \label{eq:einstein} \end{equation}\]

For easy reference within your document we are also able to create a :

As shown in Equation \@ref(eq:einstein), energy is proportional to mass.

8.3 Theorems, Lemmas, and Proofs

Bookdown supports theorem-like environments, such as theorems, lemmas, and proofs, for structured mathematical writing.

8.3.1 Adding a Theorem

Define a theorem using the following syntax:

::: {.theorem #theorem-label}
For any integer \( n \geq 1 \), the sum of the first \( n \) positive integers is given by
\[
S = \frac{n(n + 1)}{2}
\]
:::

Reference the theorem in your text:

Theorem \@ref(theorem-label) provides a formula for the sum of the first \( n \) integers.

8.3.2 Adding a Lemma

Define a lemma similarly:

::: {.lemma #lemma-label}
If \( a \) and \( b \) are even integers, then \( a + b \) is also even.
:::

Reference the lemma:

Lemma \@ref(lemma-label) confirms that the sum of two even integers is even.

8.3.3 Adding a Proof

Proofs can be added using the proof environment:

::: {.proof}
Assume \( a = 2k \) and \( b = 2m \) for integers \( k \) and \( m \). Then \( a + b = 2k + 2m = 2(k + m) \), which is even.
:::

8.4 Cross-Referencing

8.4.1 Referencing Sections

Add an ID to a section using {#label}:

## Results {#sec-results}

Reference it elsewhere:

As discussed in Section \@ref(sec-results), the findings are significant.

8.4.2 Referencing Equations, Figures, and Tables

For equations, use the \@ref() syntax with the equation’s label:

Equation \@ref(eq:einstein) demonstrates the relationship between energy and mass.

For figures and tables, the syntax is similar. Ensure the code chunk has a label:


``` r
plot(cars)

Figure 8.1: A sample figure 

See Figure \@ref(fig:figure-label) for the graph.

8.5 Tips for LaTeX in Bookdown

  1. Use Labels Consistently: Use meaningful and unique labels for cross-referencing.

  2. Use Math Mode: Always enclose mathematical symbols in $...$ or $$...$$ to render correctly.

  3. Add Theorem Styles: Customize theorem environments in _bookdown.yml for specific needs:

    theorem:
  lab: "Theorem "
  lem: "Lemma "

  4. Install Necessary Packages: Ensure you have a LaTeX distribution installed (e.g., TinyTeX).

This tutorial should provide a strong foundation for using LaTeX in your Bookdown projects. With these tools, you can create professional, structured, and highly readable documents.
